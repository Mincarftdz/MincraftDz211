# 🤖 Minecraft Wandering Bot

بوت ماين كرافت يتجول تلقائياً ويعيد الاتصال عند الطرد أو الخطأ.

## ⚙️ الإعداد

افتح ملف `config.json` وعدّل الإعدادات:

```json
{
  "server": {
    "host": "YOUR_SERVER_IP",   ← عنوان IP أو دومين السيرفر
    "port": 25565,              ← المنفذ (الافتراضي 25565)
    "version": "1.20.1"        ← إصدار السيرفر
  },
  "bot": {
    "username": "BotPlayer",   ← اسم البوت
    "auth": "offline"          ← offline = كراكد | microsoft = أصلي
  },
  "settings": {
    "reconnectDelay": 5000,    ← وقت إعادة الاتصال بالمللي ثانية
    "wanderRadius": 30,        ← نصف قطر التجول (بلوكات)
    "wanderInterval": 10000    ← فترة التجول (بالمللي ثانية)
  }
}
```

## 🚀 التشغيل

```bash
npm install
npm start
```

## 📋 أوامر الدردشة

| الأمر | الوظيفة |
|-------|---------|
| `!pos` | يعرض موقع البوت الحالي |
| `!status` | وقت التشغيل وعدد الاتصالات |
| `!stop` | إيقاف البوت |
| `!help` | قائمة الأوامر |

## ✨ المميزات

- ✅ يتجول عشوائياً بدون توقف

- ✅ إعادة اتصال عند أي خطأ
- ✅ إحياء تلقائي عند الوفاة
- ✅ تسجيل مفصّل باللون في الكونسول
